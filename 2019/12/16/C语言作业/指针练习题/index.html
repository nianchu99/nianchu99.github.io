2 、如下函数: merge(int a[] ， int n ， int b[] ， int m ， int *c) 是将两个从小到大有序数组， a 和 b 复制合并出一个有序整数序列 c ，其中形参 n 和 m 分别是数组 a 和 b 的元素个数 . 将函数补充完整并实现在主程序中的调用。
