time()模块

时间戳

时间间隔是以秒为单位的浮点小数。

每个时间戳都以自从1970年1月1日午夜（历元）经过了多长时间来表示。

时间元组

序号 属性 值
0 tm_year 2008
1 tm_mon 1到12
2 tm_mday 1到31
3 tm_hour 0到23
4 tm_min 0到59
5 tm_sec 0到61
6 tm_wday 0到6(周一是0)
7 tm_yday 1到366
8 tm_isdst -1,0,1,-1是决定是否为夏令时的旗帜
